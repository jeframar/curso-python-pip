# Curso Python Básico

Curso estructurado en 10 módulos para aprender Python desde cero.
Cada módulo incluye teoría, ejercicios guiados y retos para practicar.

## Estructura del curso

| Módulo | Tema | Conceptos clave |
|--------|------|-----------------|
| 01 | [Introducción a Python](modulo_01_introduccion/README.md) | variables, print, comentarios |
| 02 | [Tipos de datos](modulo_02_tipos_de_datos/README.md) | int, float, str, bool, type(), casting |
| 03 | [Operadores y expresiones](modulo_03_operadores/README.md) | aritméticos, comparación, lógicos, precedencia |
| 04 | [Strings y entrada del usuario](modulo_04_strings_entrada/README.md) | input(), f-strings, métodos de cadenas |
| 05 | [Condicionales](modulo_05_condicionales/README.md) | if / elif / else, operador ternario |
| 06 | [Bucles](modulo_06_bucles/README.md) | while, for, range(), break, continue |
| 07 | [Funciones](modulo_07_funciones/README.md) | def, return, scope, built-ins, lambda |
| 08 | [Listas y tuplas](modulo_08_listas_tuplas/README.md) | indexado, slicing, métodos, comprehension |
| 09 | [Diccionarios y conjuntos](modulo_09_diccionarios_conjuntos/README.md) | dict, set, operaciones de conjuntos |
| 10 | [Proyecto final](modulo_10_proyecto_final/README.md) | Agenda de tareas — integración de todos los módulos |

## Cómo usar el curso

Cada módulo contiene tres archivos:

- `README.md` — explicación teórica con ejemplos
- `ejercicios.py` — código guiado listo para ejecutar y leer
- `retos.py` — desafíos para resolver por tu cuenta

### Flujo recomendado por módulo

1. Lee el `README.md` del módulo
2. Ejecuta y estudia `ejercicios.py`
3. Resuelve los retos en `retos.py`
4. Pasa al siguiente módulo

### Ejecutar un módulo

```sh
python modulo_01_introduccion/ejercicios.py
```

## Requisitos

- Python 3.8 o superior
- No se requieren librerías externas
